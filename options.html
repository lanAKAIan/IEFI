Fresh Browser

By default, ingress.com/intel will load a full world view when it loads in a fresh browser. Setting to "Geo-Location" will attempt to detect your current location and load a view centered there. Setting to "Saved View" will attempt to load your selected saved view.
NOTE: "Geo-location" requires additional extension permissions.

New Page Load

If set to saved view, the ingress.com/intel default use of a cached value from last page load will be overridden with the selected view.

Saved View

The view that will be used for the "Fresh Browser" and "New Page Load" options.

Zoom Level

For the "Geo-Location" options, this is the default zoom level the view will load with. Higher numbers indicate increased magnification.
